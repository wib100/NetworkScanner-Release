## Network Scanner 0.3c

This release adds a built-in update check and rounds it out with multilingual polish.

### Highlights

- automatic update check with daily interval
- manual "Check for updates" action in the app settings
- direct APK download from the update dialog when available
- option to ignore a specific update version
- cleaner release-note rendering in the update dialog
- full English, German, and Spanish localization for update-related messages
